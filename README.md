# CyberRakshak AI

CyberRakshak AI is a unified, AI-powered cybersecurity platform designed to detect
deepfake media (images/videos) and phishing attacks (emails, messages, URLs) in real time.
The system is accessible across both mobile and laptop devices via a web interface.

 Problem Statement
With the rapid rise of deepfake media and phishing attacks, individuals and organizations
face serious threats such as identity theft, misinformation, financial fraud, and
reputation damage. Existing solutions are either fragmented or inaccessible to common users.

 Proposed Solution
CyberRakshak AI provides a single platform that:
- Detects deepfake images and videos using AI models
- Identifies phishing content in emails, SMS, and messages
- Explains why content is suspicious
- Works seamlessly on both mobile and desktop browsers

 Core Features
- Deepfake Detection (Image & Video)
- Phishing Detection (Email, SMS, URLs)
- AI-based confidence scoring
- Cybersecurity-themed responsive UI

 Tech Stack
- Backend: Flask (Python)
- AI/ML: CNN, NLP (TF-IDF + ML)
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Deployment-ready architecture

 Social Impact
CyberRakshak AI helps combat digital fraud, misinformation, and cybercrime by empowering
users with accessible AI-driven security tools.

 Future Scope
- Browser extension
- Mobile app wrapper
- Enterprise-level API
- Real-time monitoring
